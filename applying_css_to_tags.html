Heading Tags


This is a heading1 tag <h1>: is used to define the main heading of a webpage. It represents the highest level of heading and is typically used once per page to indicate the primary topic or title of the content.

This is a heading2 tag <h2>: is used to define a subheading or secondary heading. It is typically used to indicate sections or subsections of the content.

This is a heading3 tag <h3>: is used to define a sub-subheading or tertiary heading. It is typically used to indicate further subdivisions of the content.

This is a heading4 tag <h4>: is used to define a sub-sub-subheading or quaternary heading. It is typically used for even more specific sections of the content.

This is a heading5 tag <h5>: is used to define a sub-sub-sub-subheading or quinary heading. It is typically used for very specific sections of the content.
This is a heading6 tag <h6>: is used to define a sub-sub-sub-sub-subheading or senary heading. It is typically used for the least important sections of the content.

Paragraph Tag

This is a paragraph tag <p>: is used to define a block of text. It is typically used to group related sentences together and is often used for the main content of a webpage.

preformatted Tag

 This is a preformatted text tag <pre>:  is used to define preformatted text. It is typically used to display text in a fixed-width font, preserving whitespace and line breaks.

Blockquote Tag

This is a blockquote tag <blockquote>: is used to define a section that is quoted from another source. It is typically used to display quotes or excerpts from other texts.

Italic Tag

This is a Italic Tag <i>:The tag in HTML is used to display italicized text, primarily for styling purposes, without adding any special importance or emphasis to the content.
